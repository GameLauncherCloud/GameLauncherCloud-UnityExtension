### Features

#### Login Tab
- Connect using Game Launcher Cloud API Key
- Remember login credentials
- Easy logout functionality
- Direct link to API key management

#### Build & Upload Tab
- Load and select from available apps
- Write build notes for each upload
- One-click build and upload process
- Real-time progress indication
- Automatic ZIP compression
- Support for all major platforms

#### Tips Tab
- Build size optimization tips
- Version control best practices
- Testing recommendations
- Platform-specific advice
- Incremental patch guidelines
- Upload limit monitoring

### Technical Details
- Minimum Unity version: 2020.3
- .NET Framework 4.x / .NET Standard 2.1
- REST API integration with Game Launcher Cloud backend
- Asynchronous upload with progress tracking
- Automatic error handling and user feedback

### Known Limitations
- Mobile platforms (iOS, Android) not yet supported
- Multipart upload for large files (>500MB) coming soon
- Incremental patching features in development
